## Overview

The `use-genui-summarizer` hook provides access to GenUI summarizer functionality, allowing you to summarize text with AI-powered capabilities.

## Installation

```ts tab="npm"
npx shadcn@latest add https://gencn-ui.encatch.com/r/use-genui-summarizer.json
```

```ts tab="pnpm"

```

```ts tab="yarn"

```

