## Overview

The GenUI Provider is a provider component that enables Chrome AI APIs including Summarizer, Writer, Rewriter, and Language Detector functionality across your application.

## Installation

```ts tab="npm"
npx shadcn@latest add https://gencn-ui.encatch.com/r/genui-provider.json
```

```ts tab="pnpm"
pnpm dlx shadcn@latest add https://gencn-ui.encatch.com/r/genui-provider.json
```

```ts tab="yarn"
yarn shadcn@latest add https://gencn-ui.encatch.com/r/genui-provider.json
```

```ts tab="bun"
bunx --bun shadcn@latest add https://gencn-ui.encatch.com/r/genui-provider.json
```
